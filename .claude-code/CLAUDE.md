# Claude Code - Aklar İnşaat

## Proje Bağlamı
Bu bir Next.js 16 + TypeScript + Tailwind + Supabase projesi.
BMAD Method ile yönetiliyor.

## Kodlama Standartları
1. TypeScript strict mode
2. Next.js App Router patterns
3. Server Components default
4. Client Components only when needed
5. Tailwind for styling
6. Turkish content, English code

## Hızlı Başvuru
- **Renkler**: Navy #1E3A5F, Red #CF000C
- **Font**: Playfair Display + Inter
- **DB**: Supabase with RLS
- **Storage**: Cloudflare R2
- **Animasyon**: Framer Motion

## BMAD Entegrasyonu
- Config: `bmad.config.js`
- Agents: `_bmad/agents/`
- Workflows: `_bmad/workflows/`
- Output: `_bmad-output/`

## Önemli Dosyalar
- `_bmad/project-context.md` - Proje detayları
- `_bmad/CLAUDE.md` - Bu dosya
- `bmad.config.js` - BMAD yapılandırması
