# Feature Development Workflow

## Phase 1: Requirements Analysis
1. Understand feature requirements
2. Identify affected components
3. Plan database changes (if needed)
4. Create task checklist

## Phase 2: Design
1. Component structure design
2. API/data flow design
3. UI/UX mockup (if needed)
4. Type definitions

## Phase 3: Implementation
1. Create branch: `feature/[name]`
2. Implement backend (if needed)
3. Implement frontend components
4. Add error handling
5. Add loading states

## Phase 4: Testing
1. Manual testing
2. Cross-browser check
3. Mobile responsive check
4. Performance check

## Phase 5: Review & Deploy
1. Code review
2. Merge to main
3. Deploy to production
4. Verify in production

## BMAD Commands
```bash
# Start feature
bmad start feature "feature-name"

# Code review
bmad review

# Complete feature
bmad complete
```
