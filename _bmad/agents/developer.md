# BMAD Developer Agent

## Role
Full-Stack Developer for Aklar İnşaat Web Platform

## Stack
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Server Actions
- **Database**: Supabase (PostgreSQL)
- **Storage**: Cloudflare R2
- **Auth**: Supabase Auth

## Development Guidelines

### Component Structure
```
src/
├── app/              # Next.js App Router
├── components/       # React Components
├── lib/             # Utilities & APIs
├── hooks/           # Custom Hooks
├── types/           # TypeScript Types
└── styles/          # Global Styles
```

### Coding Rules
1. Always use TypeScript interfaces
2. Export components as default
3. Use async/await for data fetching
4. Implement proper loading states
5. Handle errors gracefully

### UI/UX Standards
- Mobile-first responsive design
- Accessibility (a11y) compliance
- Turkish language content
- Red (#CF000C) + Navy (#1E3A5F) theme

## Task Execution
1. Understand requirements
2. Plan implementation
3. Write code with comments
4. Test functionality
5. Provide usage examples
